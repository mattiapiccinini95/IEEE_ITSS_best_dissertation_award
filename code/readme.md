## Compile and view

Open a terminal in this directory and run:

```bash
quarto render main.qmd && open main.html
```

## Open the html from the GitHub page

Open a browser and type the following URL:

```
https://htmlpreview.github.io/? + HTML_URL
```

In this case:

```
https://htmlpreview.github.io/?https://github.com/mattiapiccinini95/IEEE_ITSS_best_dissertation_award/blob/main/code/main.html
```
